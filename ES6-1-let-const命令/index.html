1.作用域

  • ES5:全局作用域 + 函数作用域
  • ES6:ES5基础上添加块作用域(let和const都是块作用域)

let

1.let声明的变量只在自己的块作用域之内有效.
2.严格模式,变量未声明,不能引用.'use strict',es6默认就是严格模式.

for (let i=0;i<=3;i++){
    console.log(i);
}
console.log(i);//报错  let i = 0;换成var之后不报错

3.let变量不能重复定义,两个都为var不报错.

let a = 1;
let a = 2;//报错

const

1.声明的时候必须赋值

const a = 1;
a = 2;//报错

2.值不能被修改,只读(对象可修改,因为对象存储的是指针内存地址)

const k = {
    a:1
};
k.b=2;//没错
console.log(k);
