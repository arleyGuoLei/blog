需要用到大学里面所有的学号序列，只有通过编程自己生成，以前用易语言生成的，生成一次1500+学号并输出需要接近一分钟，数据处理和存储的方式，相比python，完全就是low爆了。本次用python来生成，速度在1ms之内 - 惊到我了。

先学习记录一个学习笔记

python获取程序运行消耗时间：

方法1

import datetime
starttime = datetime.datetime.now()
# run_fun()
endtime = datetime.datetime.now()
print (endtime - starttime).seconds

方法 2

我使用了这个方法。

import time
start = int(time.time())
# run_fun()
end = int(time.time())
print('耗费时间： ' + str(end-start)+'ms')

方法3

import time
start = time.clock()
# run_fun()
end = time.clock()
print('耗费时间： ' + str(end-start)+'ms')

方法1和方法2都包含了其他程序使用CPU的时间，是程序开始到程序结束的运行时间。方法3算只计算了程序运行的CPU时间
