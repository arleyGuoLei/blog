  1. position
  • 默认为static，这个时候top，left等四个设置不会影响位置。
  • relative，在元素本来该在的位置，通过四个定位偏移属性进行偏移。
  • absolute，父元素如果为relative | absolute | fixed ，则在父元素的内部进行偏移，否则最终可以追溯到body结点。

absolute的子元素称不开relative的父元素. 父元素需要自行设置高度.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>positionGrandparent</title>
    <style type="text/css">
        .parent{
            width: 600px;
            height: 600px;
            background: red;
            position: relative;
            top: 100px;
        }
        .son{
            width: 300px;
            height: 300px;
            background: yellow;
            position: static;/*默认就是static*/
        }
        .grandson{
            width: 100px;
            height: 100px;
            background: blue;
            position: absolute;
            top: 50px;

        }
    </style>
</head>
<body>
<div class="parent">
    <div class="son">
        <div class="grandson">孙子不认爹，认大爷。</div>
    </div>
</div>
</body>
</html>

一些position的研究：

a. 当父元素定位为relative，子元素定位也为relative时，父元素的margin设置将可能被子元素改变。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>relative</title>
    <style type="text/css">
        .parent{
            position: relative;
            height: 500px;
            width: 500px;
            background: red;
        }

        .son{
            position: relative;
            height: 200px;
            width: 200px;
            background: gold;
            margin: 100px;
        }
    </style>
</head>
<body>
<div class="parent">
    <div class="son">修改son的 margin ,父元素将生成margin-top，子元素相对于父元素垂直位置不变，水平位置生成margin。</div>
</div>


</body>
</html>

b. 利用position来实现div垂直居中:父元素设置为relative,子元素设置为absolute。子元素top为：50%，margin-top：子元素高度的一半。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>positionMiddle</title>
    <style type="text/css">
        .parent{
            position: absolute;
            width: 600px;
            height: 600px;
            background: red;

        }
        .son{
            position: absolute;
            height: 200px;
            width: 200px;
            background: yellow;

            /*关键代码*/
            top: 50%;
            margin-top: -100px;
        }
    </style>
</head>
<body>
<div class="parent">
    <div class="son">垂直居中</div>
</div>
</body>
</html>

c.position的fixed的用处实践:实现全屏半透明遮帐.(或者在浏览器中固定定位,比如回到顶部)

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>positionFixed</title>
    <style type="text/css">
        .bg-block{
            position: fixed;
            z-index: 1;
            /*四个方向都设置为0,实现全屏*/
            top: 0;
            right: 0;
            bottom: 0;
            left: 0;
            background: rgba(0,0,0,.4);
        }
    </style>
</head>
<body>
    <div class="bg-block">

    </div>
</body>
</html>
