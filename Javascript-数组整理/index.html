concat 连接数组,返回连接之后的新数组

filter 检测数值元素，并返回符合条件所有元素的数组
forEach 数组每个元素都执行一次回调函数
from 通过给定的对象中创建一个数组
map 类似from,结果返回到一个新数组
indexOf 搜索数组中的元素，并返回它所在的位置
lastIndexOf ↑

join(“x”) 把数组中的所有元素转换一个字符串,用x连接

pop 删尾

sort
//return 的小于0 小到大
//>0 大到小

points.sort(function(a,b){return a-b});

points.sort(function(a,b){return b-a});
console.log(points)
// [100, 40, 25, 10, 5, 1]

copyWithin

x = a.copyWithin(在第几个位置开始插入[从0开始数],a中的第几位开始,复制到第几位结束)
PS:复制之后 x=a ,并且长度不变

var arrA = ["a","b","c","d"]
var tmp = ["x","x","x","x"]
tmp = arrA.copyWithin(2,1,4)
// a,b,b,c  从第二个位置开始插入,并且覆盖后面的成员,数组长度会被截取成和以前一样的长度

  • 改变原数组：
    pop()
    push()
    reverse()
    shift()
    sort()
    splice()
    unshift()

  • 不改变原数组：
    concat()
